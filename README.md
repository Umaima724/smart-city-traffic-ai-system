# Smart City Traffic & Emergency Response AI System

## AL-2002 Artificial Intelligence Lab - Final Project

A modular AI-based traffic management platform for smart city environments, implementing multiple AI techniques including Artificial Neural Networks (ANN), Logic/Knowledge Base reasoning, Constraint Satisfaction Problems (CSP), and Search algorithms (BFS, UCS, A*).

---

## Project Overview

This system models traffic management as an integrated AI problem. Rather than relying on a single technique, it uses multiple AI modules that are activated based on the type of request:

- **Routine traffic requests**: Route generation only
- **Emergency situations**: Priority prediction, policy validation, traffic signal control, and optimized navigation
